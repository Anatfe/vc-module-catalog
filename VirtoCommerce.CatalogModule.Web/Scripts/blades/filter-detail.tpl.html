Quick search reference
Field names
  • find all notebooks with the "apple" brand:
    • ipod brand:apple
  • where the price has no value (or is missing):
    • is:unpriced
  • where the price has any non-null value:
    • is:priced
  • where the price is in USD and between values:
    • price_usd:[100 TO 200]
  • where the price is in any currency and between values:
    • price:[100 TO 200]
  • all active "sony" products:
    • sony is:visible
  • all hidden products:
    • is:hidden
Wildcards
  • use ? to replace a single character, and * to replace zero or more characters:
    • Da?? Red*
